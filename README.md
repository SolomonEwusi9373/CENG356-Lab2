# CENG356-Lab2
CENG 356 Lab 2 source code and execution screenshots.

This repository contains the solution for **Lab 2** of CENG 356.

## Description
The program performs:
- Conversion of an 8-bit binary number to a signed decimal integer (two’s complement)
- Conversion of a 32-bit binary number to an IEEE-754 floating-point decimal value

## Files
- `CENG356_lab2.c` – C source code with comments explaining the implementation
- `screenshots/` – Execution screenshots for required test cases

## How to Run
Compile and run using GCC:
```bash
gcc CENG356_lab2.c -o lab2
./lab2
